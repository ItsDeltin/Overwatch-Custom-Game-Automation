Deltin.CustomGameAutomation.AI Class
AI settings for Overwatch.
MethodReturn TypeDescription
AddAI(AIHero hero, Difficulty difficulty, BotTeam team, [int count])bool Add AI to the game.
CalibrateAIChecking() AI checking is determined by looking for the commendation icon of players. Sometimes, this icon is missing. This fixes it.
EditAI(int slot, AIHero setToHero, Difficulty setToDifficulty)bool Edits the hero an AI is playing and the difficulty of the AI.
EditAI(int slot, AIHero setToHero)bool Edits the hero an AI is playing.
EditAI(int slot, Difficulty setToDifficulty)bool Edits the difficulty of an AI.
GetAIDifficulty(int slot, [bool noUpdate])Difficulty? Gets the difficulty of the AI in the input slot. If the input slot is not an AI, returns null. If checking an AI's difficulty in the queue, it will always return easy, or null if it is a player.
GetAIDifficultyMarkup(int scalar, string saveAt) Obtains the markup of an AI's difficulty.
GetAISlots()List<int> Gets all slots that are AI.
GetPlayerSlots()List<int> Gets all slots that are not AI.
IsAI(int slot, [bool noUpdate])bool Checks if the input slot is an AI.
RemoveAllBotsAuto()bool Removes all AI from the game.
RemoveFromGameIfAI(int slot)bool Safely removes a slot from the game if they are an AI.
Remarks
The AI class is accessed in a CustomGame object on the CustomGame.AI field.
