Deltin.CustomGameAutomation.AI.AccurateIsAI(int slot) Method
Checks if the input slot is an AI.
ParameterTypeDescription
slot
intSlot to check.
Returns
Type: bool
Returns true if slot is AI.
ExceptionThrown
InvalidSlotExceptionThrown if slot is not a valid slot.
Remarks
IsAI() is faster but requires calling CalibrateAIChecking() beforehand. However this is more accurate and does not require calling CalibrateAIChecking().
