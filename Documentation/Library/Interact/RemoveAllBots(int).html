Deltin.CustomGameAutomation.Interact.RemoveAllBots(int slot) Method
Removes all AI from the game. Input slot must be an AI.
ParameterTypeDescription
slot
intSlot of an AI.
Returns
Type: bool
Returns true if removing all AI is successful.
ExceptionThrown
InvalidSlotExceptionThrown if slot is not a valid slot.
Remarks
slot must be the slot of an AI. If you do not know if the slot will be an AI, use AI.RemoveAllBotsAuto().
