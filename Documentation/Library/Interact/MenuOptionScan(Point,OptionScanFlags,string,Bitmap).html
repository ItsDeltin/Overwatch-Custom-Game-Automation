Deltin.CustomGameAutomation.Interact.MenuOptionScan(Point scanLocation, OptionScanFlags flags, string savelocation, Bitmap markup) Method
Scans for an option at the specified point.
ParameterTypeDescription
scanLocation
PointThe location to scan at.
flags
OptionScanFlagsThe flags for scanning.
savelocation
stringThe location to save the markup of the scanned options. Set to null to ignore.
markup
BitmapThe markup to scan for. Set to null to ignore.
Returns
Type: object

Returns a bool determining if the option is found if markup is not null and flags has the OptionScanFlags.ReturnFound flag.

Returns the location of the option if markup is not null and flags has the OptionScanFlags.ReturnLocation flag.
