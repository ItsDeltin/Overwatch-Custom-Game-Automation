Deltin.CustomGameAutomation.Interact.MenuOptionScan(int slot, Bitmap markup, int minimumPercent, int max, [double yincrement]) Method
Scans a player menu for an option.
ParameterTypeDescription
slot
intSlot's menu to scan.
markup
BitmapBitmap markup of option to scan for.
minimumPercent
intMinimum percent the markup has to match an option in the menu.
max
intMaximum options to scan.
yincrement
doubleAmount to skip on Y axis after every markup scan.
Returns
Type: bool
Returns true if the option in the markup has been found, else returns false.
ExceptionThrown
InvalidSlotExceptionThrown if slot is not a valid slot.
