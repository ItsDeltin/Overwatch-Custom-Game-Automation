Deltin.CustomGameAutomation.OptionScanFlags Enumerator
Flags for scanning an option menu in Overwatch.
ValueDescription
None No flags.
OpenMenu Open the menu before scanning starts.
CloseMenu Close the menu after scanning finishes.
CloseIfNotFound Close the menu if the option being scanned for is not found.
Click Click the option if it is found.
ReturnLocation Return the location of the option.
ReturnFound Return whether or not the option is found.
