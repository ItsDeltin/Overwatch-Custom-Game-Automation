Deltin.CustomGameAutomation.Pause Class
Controls Overwatch's pause feature.
MethodReturn TypeDescription
IsPaused()bool Determines if the game is paused.
PauseGame() Pauses the game.
TogglePause() Toggle pause.
UnpauseGame() Unpauses the game.
Remarks
The Pause class is accessed in a CustomGame object on the CustomGame.Pause field.
