CustomGame.GetCurrentOverwatchEvent()
Syntax 
public Event GetCurrentOverwatchEvent()
Deltin.CustomGameAutomation.Event
Returns the current Overwatch event depending on the date.

Gets the current Overwatch event.

The event returned depends on the current date, based off of past Overwatch events. Specific times may be inaccurate. Setting CustomGame.CurrentOverwatchEvent is required when using ToggleMap() because of event-exclusive maps. Dates are inclusive.

Event.HalloweenTerror: October 10th - November 1st
