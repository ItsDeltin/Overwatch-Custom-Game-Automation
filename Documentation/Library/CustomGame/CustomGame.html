Deltin.CustomGameAutomation.CustomGame Class
Automates Overwatch's custom games.
ConstructorDescription
CustomGame([IntPtr overwatchHandle], [ScreenshotMethod screenshotMethod], [bool openChatIsDefault]) Creates new CustomGame object using an Overwatch process.
MethodReturn TypeDescription
CreateOverwatchProcess(OverwatchProcessInfo processInfo)IntPtr Creates a new Overwatch process.
DisableOverwatchWindowInput() Disables input for the Overwatch window.
Dispose() Disposes of all resources being used by the CustomGame instance.
EnableOverwatchWindowInput() Enables input for the Overwatch window after disabling it with .
GetCurrentOverwatchEvent()Event Gets the current Overwatch event. This compares the current date with past event's start and end times, so specific time may be a little off.
GetGameState()GameState Returns the state of the game. In lobby: GameState.InLobbyWaiting for players: GameState.WaitingIngame: GameState.IngameCommending players: GameState.Ending_Commend
GetInvitedCount(List<int> playersConnected)int Gets the number of players who were invited but are not ingame.
GetInvitedSlots(List<int> playersConnected)List<int> Gets a list of players who were invited but are not ingame.
InvitePlayer(string playerName, [InviteTeam team])bool Invites a player to the game via battletag.
IsSlotBlue(int slot)bool Returns true if the slot is in blue.
IsSlotInQueue(int slot)bool Returns true if the slot is in queue.
IsSlotRed(int slot)bool Returns true if the slot is in red.
IsSlotSpectator(int slot)bool Returns true if the slot is in Spectator.
IsSlotValid(int slot)bool Returns true if the slot is blue, red, spectator, or in queue.
RestartGame() Restarts the game.
SaveScreenshot(string path) Takes a screenshot of the Overwatch window and saves it as an image to the specified path.
SendServerToLobby() Sends server to the lobby.
SetHeroRoster(ToggleAction ta, BotTeam team, Hero[] heroes) Toggles what heroes can be selected.
SetHeroSettings(SetHero[] herodata) Change individual hero settings.
SetupOverwatchWindow() Positions the Overwatch window to be usable by the CustomGame class.
StartGame() Starts the game.
StartGamemode() Starts the game when waiting for players.
ToggleMap(ToggleAction ta, Map[] maps) Toggles maps in Overwatch.
ToggleMap(ModesEnabled modesEnabled, Event currentOverwatchEvent, ToggleAction ta, Map[] maps) Toggles maps in Overwatch.
WaitForSlotUpdate([int maxtime])bool Waits for the slots in overwatch to change.
