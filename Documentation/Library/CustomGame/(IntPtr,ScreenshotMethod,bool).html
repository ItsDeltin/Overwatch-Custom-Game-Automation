Deltin.CustomGameAutomation.CustomGame([IntPtr overwatchHandle], [ScreenshotMethod screenshotMethod], [bool openChatIsDefault]) Constructor
Creates new CustomGame object using an Overwatch process.
ParameterTypeDescription
overwatchHandle
IntPtrOverwatch process handle to use. Leave at default to use the first Overwatch process found.
screenshotMethod
ScreenshotMethodMethod to take screenshots with.
openChatIsDefault
boolDetermines if the chat should be opened at all times. Command scanning is more reliable if true.
ExceptionThrown
MissingOverwatchProcessExceptionThrown if no Overwatch processes were found.
