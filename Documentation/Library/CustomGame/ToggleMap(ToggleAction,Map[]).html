Deltin.CustomGameAutomation.CustomGame.ToggleMap(ToggleAction ta, Map[] maps) Method
Toggles maps in Overwatch.
ParameterTypeDescription
ta
ToggleActionDetermines if all maps should be enabled, disabled or neither before toggling.
maps
Map[]Maps that should be toggled.
ExceptionThrown
ArgumentNullExceptionThrown if CustomGame.ModesEnabled is null.
Remarks
CustomGame.ModesEnabled must be set to use this method. CustomGame.CurrentOverwatchEvent must be set if a seasonal Overwatch event is on.

All maps:
Examples
The code below will disable all maps but Hanamura, Gibraltar, and Ilios. 
using Deltin.CustomGameAutomation;
            
public class ToggleMapExample
{
    public static void SetEnabledMaps(CustomGame cg) 
    {
        cg.ModesEnabled = new ModesEnabled()
        {
            Assault = true,
            AssaultEscort = true,
            Control = true,
            Escort = true
        }
        cg.CurrentOverwatchEvent = Event.None;
        cg.ToggleMap(ToggleAction.DisableAll, Map.A_Hanamura, E_Gibraltar, C_Ilios);
    }
}
