Deltin.CustomGameAutomation.CustomGame.ToggleMap(ModesEnabled modesEnabled, Event currentOverwatchEvent, ToggleAction ta, Map[] maps) Method
Toggles maps in Overwatch.
ParameterTypeDescription
modesEnabled
ModesEnabledThe modes enabled in the overwatch game.
currentOverwatchEvent
EventThe current Overwatch event.
ta
ToggleActionDetermines if all maps should be enabled, disabled or neither before toggling.
maps
Map[]Maps that should be toggled.
ExceptionThrown
ArgumentNullExceptionThrown if modesEnabled is null.
Remarks

All maps:
Examples
The code below will disable all maps but Hanamura, Gibraltar, and Ilios. 
using Deltin.CustomGameAutomation;
            
public class ToggleMapExample
{
    public static void SetEnabledMaps(CustomGame cg) 
    {
        cg.ToggleMap(
            new ModesEnabled() 
            {
                Assault = true,
                AssaultEscort = true,
                Control = true,
                Escort = true
            },
            Event.None,
            ToggleAction.DisableAll, 
            Map.A_Hanamura, E_Gibraltar, C_Ilios);
    }
}
