CustomGame.CG_Maps.MapNameFromID(Map map)
Syntax 
public string MapNameFromID(
    Map map
)
map
Deltin.CustomGameAutomation.Map
Map ID to get name from.
System.String
The name of the map.

Gets map name from map ID.

The code below will return the string "A_TempleOfAnubis".

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    CustomGame cg = new CustomGame();
    Map map = Map.A_TempleOfAnubis; // Map ID of A_TempleOfAnubis
    Console.WriteLine(cg.Maps.MapNameFromID(map)); // Returns string "A_TempleOfAnubis"
}
