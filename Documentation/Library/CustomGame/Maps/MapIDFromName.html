CustomGame.CG_Maps.MapIDFromName(string map)
Syntax 
public Map MapIDFromName(
    string map
)
map
System.String
Map name to get ID from.
Deltin.CustomGameAutomation.Map
The ID of the map from the name.

Gets map ID from string.

The example below will switch the current map to Hollywood.

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    CustomGame cg = new CustomGame();

    // Set enabled modes.
    cg.ModesEnabled.Assault = true;
    cg.ModesEnabled.AssaultEscort = true;
    cg.ModesEnabled.Control = true;
    cg.ModesEnabled.Escort = true;

    // Set the current overwatch event.
    cg.CurrentOverwatchEvent = cg.GetCurrentOverwatchEvent();

    string mapname = "AE_Hollywood"; // Map name
    Map mapID = cg.Maps.MapIDFromName(mapname); // Get map ID from mapname
    cg.Maps.ToggleMap(MapToggleAction.DisableAll, mapID); // Disable all maps except Hollywood
}
