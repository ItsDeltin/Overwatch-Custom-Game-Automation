CustomGame.CG_Pause.TogglePause()
Syntax 
public void TogglePause()

Toggles the moderator pause feature. Apon unpausing, a 5 second countdown is shown to all players until they can move again.

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    // 5 second countdown for race.

    CustomGame cg = new CustomGame();

    cg.Pause.TogglePause(); // Pause game.
    cg.Pause.TogglePause(); // Unpause and start 5 second countdown.
    cg.Chat.Chat("Get ready to race!");
}
