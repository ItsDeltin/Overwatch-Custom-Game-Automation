CustomGame.CG_Settings.SetMaxPlayers(int? blueCount, int? redCount, int? ffaCount, int? spectatorCount)
Syntax 
public void SetMaxPlayers(
    int? blueCount,
    int? redCount,
    int? ffaCount,
    int? spectatorCount
)
blueCount
System.Int32?
Maximum number of blue players. Must be in the range of 1-6. Set to null to ignore.
redCount
System.Int32?
Maximum number of red players. Must be in the range of 1-6. Set to null to ignore.
ffaCount
System.Int32?
Maximum number of FFA players. Must be in the range of 1-12. Set to null to ignore.
spectatorCount
System.Int32?
Maximum number of spectators. Must be in the range of 0-12. Set to null to ignore.

Sets maximum player count of blue team, red team, FFA count or spectators.

using System;
