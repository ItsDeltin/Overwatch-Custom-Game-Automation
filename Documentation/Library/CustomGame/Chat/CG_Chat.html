Deltin.CustomGameAutomation.CustomGame.CG_Chat class

Method Description
Chat(string) Sends a message to the chat ingame.
JoinChannel(Channel) Join chat channel.
LeaveChannel(Channel) Leaves a chat channel so no chat messages can be sent or recieved on that channel.
Field Description
string ChatPrefix Adds a prefix to chat messages sent by Chat().
bool BlockGeneralChat Blocks sending chat messages to the general channel.
