CustomGame.CG_Chat.SwapChannel(Channel channel)

public void SwapChannel(
    Channel channel
)
channel
Deltin.CustomGameAutomation.Channel
Channel to swap to.

Switches to a chat channel.

The code below will send a message to everyone in spectators, then join match channel.

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    CustomGame cg = new CustomGame();

    cg.Chat.SwapChannel(Channel.Team); // join team channel (spectator)

    cg.Chat.Chat("Welcome to typical boss battle! You are currently in the queue, please wait.");

    cg.Chat.SwapChannel(Channel.Match); // join match channel
}
