Deltin.CustomGameAutomation.Commands class

Class Description
CommandData Data regarding an executed command.
Method Description
RemoveAllExecutedCommands() Disposes data of all executed commands.
RemoveExecutedCommand(int) Disposes data of an executed command.
Property Description
ReadOnlyCollection<CommandData> ExecutedCommands List of executed commands.
Field Description
bool AllowMultipleCommandExecutions Allows one executor to execute more than one command.
bool Listen Set to true to start listening for commands. Set to false to stop.
Dictionary<string, bool> ListenTo Dictionary of commands to listen to. Indexer is the command to listen to, value is a bool that determines if this command should be listened to.
bool ListenToAllCommands If true, listens to all commands rather than just the commands in the ListenTo dictionary.
bool SameExecutorCommandUpdate If true, updates an already executed command from an executor. AllowMultipleCommandExecutions must be false.
