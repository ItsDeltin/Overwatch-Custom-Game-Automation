Deltin.CustomGameAutomation.CustomGame.InvitePlayer(string playerName, int slot) Method
Invites a player to the game via battletag.
ParameterTypeDescription
playerName
stringBattletag of the player to invite. Is case sensitive. Ex: Tracer#1818
slot
intSlot that the invited player will join.
Returns
Type: bool
Returns true if playerName is a valid battletag.
ExceptionThrown
ArgumentNullExceptionThrown if playerName is null.
