CustomGame.CG_Interact.SwapTeam(int slot)
Syntax 
public bool SwapTeam(
    int slot
)
slot
System.Int32
Slot to swap team.
System.Boolean
Returns true if swapping team of slot was successful.

Swaps the team of a player. Swapping a blue player will swap their team to red. A red player will swap swap to blue. A player queueing for blue team will be swapped to queueing for red and vice versa.
