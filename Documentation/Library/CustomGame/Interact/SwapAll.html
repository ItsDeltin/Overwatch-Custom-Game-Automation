CustomGame.CG_Interact.SwapAll()
Syntax 
public void SwapAll()

Swaps players in both teams. Only works when the max player count is the same for both teams.

The example below will swap the players in both teams then restart the game during commending.

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    // Swap players on both teams at end of game to prevent defend/attack rotation.

    CustomGame cg = new CustomGame();

    while(true)
    {
        if (cg.GetGameState() == GameState.Ending_Commend)
        {
            cg.Interact.SwapAll();
            cg.RestartGame();
        }
    }
}
