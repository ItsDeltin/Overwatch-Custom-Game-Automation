Deltin.CustomGameAutomation.Settings Class
Custom Game settings in Overwatch.
MethodReturn TypeDescription
LoadPreset(int preset)bool Loads a preset saved in Overwatch.
SetGameName(string name) Changes the custom game's name.
SetJoinSetting(Join setting) Changes who can join.
SetMaxPlayers(int? blueCount, int? redCount, int? ffaCount, int? spectatorCount) Sets the max player count for blue team, red team, free for all, or spectators.
SetSettings(GameSettings settings) Changes settings in the Custom Game.
SetTeamName(Team team, string name) Changes a team's name.
Remarks
The Settings class is accessed in a CustomGame object on the CustomGame.Settings field.
