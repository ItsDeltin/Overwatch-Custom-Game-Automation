Deltin.CustomGameAutomation.Settings.SetMaxPlayers(int? blueCount, int? redCount, int? ffaCount, int? spectatorCount) Method
Sets the max player count for blue team, red team, free for all, or spectators.
ParameterTypeDescription
blueCount
int?Maximum number of blue players. Must be in the range of 1-6. Set to null to ignore.
redCount
int?Maximum number of red players. Must be in the range of 1-6. Set to null to ignore.
ffaCount
int?Maximum number of FFA players. Must be in the range of 1-12. Set to null to ignore.
spectatorCount
int?Maximum number of spectators. Must be in the range of 0-12. Set to null to ignore.
ExceptionThrown
ArgumentOutOfRangeExceptionThrown if blueCount, redCount, ffaCount, or spectatorCount is less than 0 or greater than their max values.
