Deltin.CustomGameAutomation.Settings.SetTeamName(Team team, string name) Method
Changes a team's name.
ParameterTypeDescription
team
TeamTeam to change name.
name
stringName to change team's name to.
ExceptionThrown
ArgumentOutOfRangeExceptionThrown if name is less than 1 character or greater than 15 characters. Also thrown if team is Spectator or Queue.
ArgumentExceptionThown if name has the text "admin" in it.
ArgumentNullExceptionThrown if name is null.
