Deltin.CustomGameAutomation.OverwatchProcessInfoAuto Class
Data for creating an Overwatch process automatically.
ConstructorDescription
OverwatchProcessInfoAuto() Data for creating an Overwatch process automatically.
FieldTypeDescription
AutomaticallyCreateCustomGamebool If true, the Overwatch process will automatically create a Custom Game.
BattlenetExecutableFilePathstring The path to the battle.net executable. Defaults to "C:\Program Files (x86)\Blizzard App\Battle.net.exe"
CloseOverwatchProcessOnFailurebool Closes the Overwatch process if it fails to log in.
MaxBattlenetStartTimeint The maximum amount of time to wait for Battle.net to start.
MaxOverwatchStartTimeint The maximum amount of time to wait for Overwatch to start.
MaxWaitForMenuTimeint The maximum amount of time to wait for the menu to load.
OverwatchSettingsFilePathstring The path to Overwatch's settings file. Defaults to "C:\Users\(EnvironmentName)\Documents\Overwatch\Settings\Settings_v0.ini"
ScreenshotMethodScreenshotMethod The method that is used to take screenshots of the Overwatch window.
