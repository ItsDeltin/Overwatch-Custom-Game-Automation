Deltin.CustomGameAutomation.PlayerInfo.PlayerExists(string battletag) Method
Checks if a player account exists via battletag. Is case sensitive.
ParameterTypeDescription
battletag
stringBattletag of player to check. Is case sensitive.
Returns
Type: bool
Returns true if player exists, else returns false.
ExceptionThrown
