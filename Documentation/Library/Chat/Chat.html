Deltin.CustomGameAutomation.Chat Class
Chat for Overwatch.
MethodReturn TypeDescription
JoinChannel(Channel channel) Joins a channel after leaving it with LeaveChannel.
LeaveChannel(Channel channel) Leaves a channel so no chat messages can be sent or recieved on that channel.
SendChatMessage(string text) Send message to chat.
SwapChannel(Channel channel) Swaps to a chat channel.
Remarks
The Chat class is accessed in a CustomGame object on the CustomGame.Chat field.
