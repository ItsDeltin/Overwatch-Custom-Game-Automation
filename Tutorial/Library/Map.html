To set maps, you need to set CustomGame.EnabledModes and CustomGame.CurrentOverwatchEvent. You can get the current Overwatch event using CustomGame.GetCurrentOverwatchEvent(), while the enabled modes need to be set manually.

CustomGame cg = new CustomGame();

// Set the enabled modes.
cg.ModesEnabled.Assault = true;
cg.ModesEnabled.AssaultEscort = true;
cg.ModesEnabled.Control = true;
cg.ModesEnabled.Escort = true;

// Set the current event.
cg.CurrentOverwatchEvent = Event.HalloweenTerror; // You can do it manually, or...
cg.CurrentOverwatchEvent = cg.GetCurrentOverwatchEvent(); // Get it automatically. This makes it so you don't need to update your code every time an Event starts or ends.
