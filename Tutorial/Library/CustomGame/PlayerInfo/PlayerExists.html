CustomGame.CG_PlayerInfo.PlayerExists(string battletag, params Region[] regions);
Syntax 
public bool PlayerExists(
    string battletag, 
    params Region[] regions
)

battletag
Battletag of user to check exists.
Type: System.String

regions
Regions to check user.
Type: Deltin.CustomGameAutomation.Region

Type: System.Boolean
Returns true if the player exists.

Checks if a player account exists by their battletag. The user must play one full game in an input region to be found.
