CustomGame.CG_AI.GetAIDifficulty(int slot)
Syntax 
public Difficulty? GetAIDifficulty(
    int slot
)

slot
Slot to get difficulty from.
Type: int

Type: Deltin.CustomGameAutomation.Difficulty?
Returns the difficulty of the slot's AI. Returns Difficulty.Easy if the slot is an AI in queue. Returns null if the slot is a player or empty.
