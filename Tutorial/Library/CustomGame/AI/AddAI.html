CustomGame.CG_AI.AddAI(AIHero hero, Difficulty difficulty, BotTeam team, int count = -1);
Syntax 
public bool AddAI(
    AIHero hero, 
    Difficulty difficulty, 
    BotTeam team, 
    int count = -1
)

hero
AI hero type to add.
Type: Deltin.CustomGameAutomation.AIHero

difficulty
Difficulty of AI added.
Type: Deltin.CustomGameAutomation.Difficulty

team
Team that the AI is added to.
Type: Deltin.CustomGameAutomation.BotTeam

count
Number of AI that is added. Defaults to -1. Adds maximum amount of bots possible when at -1.
Type: System.Int32

Type: System.Boolean
Returns true if AI was added successfully. Returns false if no more AI can be added.

Adds AI to the game. Returns false if fails to add AI, returns true if it successfully adds AI.

The example below will add 6 hard recommended AI to the blue team and 4 hard Bastions to the red team.

using System;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    CustomGame cg = new CustomGame();

    cg.AI.AddAI(AIHero.Recommended, Difficulty.Hard, BotTeam.Blue); // Add 6 hard recommended heroes to blue team

    cg.AI.AddAI(AIHero.Bastion, Difficulty.Hard, BotTeam.Red, 4); // Add 4 hard bastions heroes to red team
}
