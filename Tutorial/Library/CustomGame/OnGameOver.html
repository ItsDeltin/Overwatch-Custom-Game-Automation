CustomGame.OnGameOver

OnGameOver is an event that gets executed when the Overwatch game ends.

To get the team that wins, the blue team must have \ (backslash) on the start of their name, and the red team must have * (asterisk) on the start of their name.

The example below will send a message to chat when the game is over.

class Program
{
    static void Main(string[] args)
    {
        CustomGame cg = new CustomGame();

        cg.GameSettings.SetTeamName(PlayerTeam.Blue, @"\ Blue Team");
        cg.GameSettings.SetTeamName(PlayerTeam.Red,   "* Red Team");

        cg.StartGamemode();

        cg.OnGameOver += GameOver;
    }

    static void GameOver(object sender, GameOverArgs e)
    {
        CustomGame cg = (CustomGame)sender;
        cg.Chat.Chat(The game is over! Team " + e.GetWinningTeam() +  has won!");
    }
}
