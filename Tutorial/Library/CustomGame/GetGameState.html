CustomGame.GetGameState();
Syntax 
public GameState GetGameState()
Type: Deltin.CustomGameAutomation.GameState
Returns the current state of the game.

Returns a Deltin.CustomGameAutomation.GameState enum of the current state of the game.

In lobby: GameState.InLobby
Waiting For Players: GameState.Waiting
Playing POTG: GameState.Ending_POTG
Commending: GameState.Ending_Commend
Unknown: GameState.Unknown

The example below will restart the game after POTG.

using System;
using System.Threading;
using Deltin.CustomGameAutomation;

...

static void Main(string[] args)
{
    // Restart the game during commending.

    CustomGame cg = new CustomGame();

    while(true)
    {
        if (cg.GetGameState() == GameState.Ending_Commend)
            cg.RestartGame();
        Thread.Sleep(1000);
    }
}
