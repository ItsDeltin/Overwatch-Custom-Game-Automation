CustomGame.CG_Interact.RemoveAllBots(int slot)
Syntax 
public bool RemoveAllBots(
    int slot
)

slot
Slot of an AI in the game.
Type: int

Type: System.Boolean
Returns true if removing all AI was successful.

Removes all AI from the game. Input slot must be an AI.

Unless you know the input slot will definitely be an AI, use CustomGame.CG_AI.RemoveAllBotsAuto().
