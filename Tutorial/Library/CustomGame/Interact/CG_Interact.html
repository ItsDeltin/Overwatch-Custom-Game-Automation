Deltin.CustomGameAutomation.CustomGame.CG_Interact class

Method Description
Move(int, int) Moves player's slot.
RemoveAllBots(int) Removes all AI from the game. Input slot must be an AI.
RemoveFromGame(int) Removes player or AI from the game.
SwapAll() Swaps players in both teams. Only works when the max player count is the same for both teams.
SwapTeam(int) Swaps team of player.
SwapToBlue(int) Swaps player to blue team.
SwapToNeutral(int) Swaps player in queue to neutral team.
SwapToRed(int) Swaps player to red team.
SwapToSpectators(int) Swaps player to spectators.
