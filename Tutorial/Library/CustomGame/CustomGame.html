Deltin.CustomGameAutomation.CustomGame class

Constuctor Description
CustomGame(IntPtr overwatchHandle = new IntPtr()) Creates new CustomGame object using an Overwatch processes main window handle.
Class Description
CG_AI Fields relating to AI. Accessed in a CustomGame object under the field CustomGame.AI.
CG_Chat Fields relating to the Overwatch chat. Accessed in a CustomGame object under the field CustomGame.Chat.
CG_Settings Fields related to the custom game settings. Accessed in a CustomGame object under the field CustomGame.GameSettings.
CG_Interact Fields related to player roles. Accessed in a CustomGame object under the field CustomGame.Interact.
CG_Maps Fields related to maps. Accessed in a CustomGame object under the field CustomGame.Maps.
CG_Pause Fields related to the moderator pause feature. Accessed in a CustomGame object under the field CustomGame.Pause.
Commands Fields related to chat commands. Accessed in a CustomGame object under the field CustomGame.Command.
Method Description
GetCurrentOverwatchEvent() Gets the current event in Overwatch.
GetGameState() Gets the current state of the game.
GetInvitedCount(List<int>) Returns the number of players that are invited but not ingame.
GetInvitedSlots(List<int>) Returns a list of the slots that are invited but not ingame.
InvitePlayer(string, InviteTeam) Invites a player to the custom game. Is case sensitive.
RestartGame() Restarts the game.
SendServerToLobby() Sends the custom game server to the lobby.
SetHeroRoster(ToggleAction, BotTeam, Hero[]) Changes hero roster.
SetHeroSettings(SetHero[]) Changes individual hero settings.
StartGame() Starts the game from the lobby.
StartGamemode() Starts the game when waiting for players.
Property Description
int BlueCount Number of players in blue team.
List<int> BlueSlots List of the slots that are filled on blue team.
int PlayerCount Number of players in blue and team.
List<int> PlayerSlots List of the slots that are filled on blue and red team.
int QueueCount Number of players in queue.
int RedCount Number of players in red team.
List<int> RedSlots List of the slots that are filled on red team.
int SpectatorCount Number of players in spectators.
List<int> SpectatorSlots List of the slots that are filled on spectators.
int TotalPlayerCount Number of players in custom game server, including players invited.
List<int> TotalPlayerSlots List of slots filled in custom game server, including players invited.
Field Description
CG_AI AI
CG_Chat Chat
CG_Settings GameSettings
CG_Interact Interact
CG_Maps Maps
CG_Pause Pause
CG_PlayerInfo PlayerInfo
Event CurrentOverwatchEvent The current Overwatch event.
ModesEnabled ModesEnabled The modes enabled in custom games.
Event Description
OnGameOver Events that are executed when the game is over.
